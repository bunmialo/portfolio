Weather App

Bunmi's Weather App is a web app created using the weather API that allows users to search for weather updates by zip codes. Upon hitting the submit button, users can see the name of the location, main temperature, the weather condition, and the minimum and maximum temperatures.

Trivia Game App

Inspired by the popular game show "Who Wants To Be a Millionaire," Who Wants To Be a Thousandnaire is a trivia game that was built using the javascript Document Object Model (DOM) manipulation, which allows users to answer questions and get an incentive of a thousand for each correctly answered question. Users become "Thousandnaire" upon successfully completing the game, or have the option to start over if not all questions are answered correctly.

Country List App

Bunmi's List is a fullstack Express.js web app deployed to Heroku that shows a list of countries and some of their distinctive features in the app (SQL) database, that includes the country's capital, currency, a language they speak, a food type, and so forth.
Users are prompted to login or register in order to explore app. Upon registering/logging in, users can add countries of their choice to this list, edit and also delete their choice.

Movie Search App

Generated using React frontend and Ruby-on-Rails backend, bOOmlist is an awesome web app deployed to Heroku that allows users to view a list of popular movies in the bOOmlist (SQL) database and also search for the current popular movies available online from The Movie Database.
Users are able to add movies from their search results to database, edit and also delete them.
